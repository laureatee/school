Командная строка.
1. Чтбы запусть команду, надо набрать несколько слов и нажать enter
2. Первое слово называется конандной программы, остальные - аргументами командной строки
Например: py и hello.py
Здесь программа - интерпритатор питон (ру) 
Её первый аргумент - путь к файлу нашей программы
3. перемещение между папками происходит с помощью команды cd = change diectory
cd .. переход на уроквень выше(произносится цэдэ две точки)
cd 20251007_if_prac
чтобы запустить программу нужно находится в самом файле и написать py ....(название файла.ру)
4. Посмотреть список файлов и папок в директории
